## 一、启动与停止

```bash
# 进入论坛目录
cd D:\cld-ai2\forum

# 启动（生产模式）
npm start

# 启动（开发模式，代码改动自动重启）
npm run dev

# 停止：在终端按 Ctrl + C
```

启动后访问 `http://localhost:3000`

## 三、前台功能

### 浏览
- 首页展示六大板块及最新主题
- 点击板块名查看该分类下所有主题
- 点击主题进入详情页、查看回复

### 注册与登录
- 点击右上角「注册」创建超越者身份
- 登录后可发帖和回复

### 发帖
- 登录后点击「发帖」按钮
- 选择板块、填写标题和内容后提交

### 回复
- 进入主题详情页，底部有回复框
- 主题被锁定后无法回复（类比 WCSC 权限锁定）

## 四、管理后台（仅管理员）

访问 `http://localhost:3000/admin`

### 4.1 数据概览
- 首页显示用户数、主题数、回复数、分类数

### 4.2 板块管理
可以增删改六个板块：
| 板块 | 含义 |
|------|------|
| 情报交换 | 异常现象、WCSC内部动态等真假难辨的信息流动 |
| 委托发布 | 从寻猫到调查具名者失控的委托 |
| 知识共享 | 谜题理解、能力开发心得、准则研究 |
| 匿名社交 | 超越者唯一的自由呼吸空间 |
| 日常分享 | 普通生活内容 |
| WCSC通告 | 官方公告与高危通缉令 |

### 4.3 站点设置（`/admin/settings`）
- **站点名称**：浏览器标题和页眉显示
- **站点描述**：首页副标题
- **每页帖子数**：分页每页显示数量
- **页脚文字**：页面底部
- **允许注册**：开关新用户注册
- **自定义 CSS**：直接写 CSS，实时覆盖默认样式

### 4.4 用户管理（`/admin/users`）
- 查看所有用户列表
- 修改用户角色（普通用户 / 版主 / 管理员）
- 删除用户（不能删除自己）

│   ├── partials/

## 七、常见操作

| 操作 | 方法 |
|------|------|
| 改颜色主题 | 后台 → 站点设置 → 自定义 CSS，覆盖 CSS 变量 |
| 新增功能模块 | `routes/` 下新建路由，`views/` 下新建模板，`app.js` 挂载 |
| 备份数据 | 复制 `forum.db` 文件，这是全部数据 |
| 修改端口 | `PORT=8080 npm start` |
| 从外网访问 | 配置 Nginx 反向代理 + 开放端口 + 绑定域名 |
